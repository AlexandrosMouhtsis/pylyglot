Pylyglot {% trans "is a web based glossary compedium for Free and Open Source Software translators heavily inspired on the Open-tran.eu web site." %}

{% trans "The main objective for Pylyglot is to 'provide a concise, yet comprehensive compilation of a body of knowledge' for translators derived from existing Free and Open Source Software translations. The more data is added to the system, the more accurate and reliable the results will be!" %}
